# aerospace-config

My [AeroSpace](https://github.com/nikitabobko/AeroSpace) tiling window manager configuration for macOS.

## Installation

1. Install AeroSpace via Homebrew:

```bash
brew install --cask nikitabobko/tap/aerospace
```

2. Clone this repo:

```bash
git clone https://github.com/plivesey/aerospace-config.git ~/.config/aerospace
```

3. Restart AeroSpace (or reload config with `alt+shift+;` then `esc`).

## Highlights

### Window Management

| Keybinding | Action |
|---|---|
| `alt + h/j/k/l` | Focus left/down/up/right |
| `alt + shift + h/j/k/l` | Move window left/down/up/right |
| `alt + shift + space` | Fling window to first empty workspace |
| `alt + /` | Toggle tiles horizontal/vertical |
| `alt + ,` | Toggle accordion horizontal/vertical |
| `alt + -/=` | Resize window |
| `alt + tab` | Switch to last workspace |

### Workspace Shortcuts

`alt + <key>` to focus a workspace, `alt + shift + <key>` to move a window to it. Workspaces: `1-9`, `A-G`, `I`, `M-Z`.
